## 옵시디언 14금칙 (타겟: .md)

### ❶ div 내부 빈줄 금지
`<div>` ~ `</div>` 사이 빈 줄 1개라도 있으면 옵시디언이 HTML 블록을 끊고 마크다운 파서로 전환 → 레이아웃 붕괴.

### ❷ div 내부 마크다운 문법 금지
`**볼드**`, `*이탤릭*`, `- 리스트`, `[[위키링크]]`, `[링크](url)` 모두 금지. HTML 태그만 사용.

| 마크다운 | HTML 대체 |
|---|---|
| `**텍스트**` | `<strong>텍스트</strong>` |
| `*텍스트*` | `<em>텍스트</em>` |
| `- 항목` | `<ul><li>항목</li></ul>` |
| `[링크](url)` | `<a href="url">링크</a>` |

### ❸ 모든 텍스트 태그에 color 개별 지정
특히 다크 배경. div의 `color`는 `<p>`·`<strong>`·`<li>`·`<td>`·`<span>` 하위에 전파되지 않음. 각각 `style="color:…"` 명시.

### ❹ 다크 배경 div 내 마크다운 테이블 금지
옵시디언 테마 stripe가 배경 override. div 바깥에 배치하거나 `<table><td>`에 `background-color` 명시.

### ❺ div 블록 전후 빈줄 필수
div 시작 전·종료 후 빈줄 1개 없으면 마크다운 콘텐츠 흡수.

### ❻ 인라인 style만 사용
`<style>`·`<link>`·CSS class 전부 미지원. 모든 스타일은 태그 `style=""` 속성에.

### ❼ flex/grid는 리딩뷰 전용
편집뷰(라이브프리뷰·소스모드)에서 무너짐. 리딩뷰 확인 필수.

### ❽ font-family 지정 금지
옵시디언 기본폰트와 충돌. size/weight/color만 조정.

### ❾ margin 금지 (div 외부 간격)
예측 불가 → padding만 사용. 간격은 `<br>` 스페이서 또는 빈줄로.

### ❿ 연한 색상 함정
`#fafafa`(배경)·`#eee`(보더)·`#999`(레이블)은 라이트 테마에서 거의 안 보임. 기준: 보더 `#b0b0b0`↑, 배경 `#efefef`↑, 레이블 `#666`↑.

### ⓫ 연속 div 카드 사이 `<br>` 스페이서 필수
빈줄 1개만으로는 옵시디언이 간격 0으로 붙여버림. 카드 사이 `<br>` 스페이서.

### ⓬ `<details>` 내부 빈줄·마크다운 금지
내부에 빈줄 또는 `**볼드**` 있으면 HTML 블록 끊어져 콘텐츠가 details 바깥으로 빠짐. 내부도 HTML only.

### ⓭ `<summary>` 내 삼각형 문자 금지
`▶ ► ▸ ▹ ◀ ◄ ◂ ◃` 금지. 옵시디언 네이티브 토글 삼각형과 중복.

### ⓮ `<br>` 뒤 블록 태그 같은 줄 금지
`<br>` + 블록 태그 동일 줄 → div 스타일 전부 무시. 반드시 `<br>` 단독 줄 → 빈줄 → `<div>` 새 줄.

## 대응 치트시트

| 증상 | 원인 | 해결 |
|---|---|---|
| 카드가 투명하게 보임 | ❿ 연한 보더/배경 | 기준선 이상으로 진하게 |
| div 안 `**볼드**` 그대로 노출 | ❷ 마크다운 문법 | `<strong>`으로 교체 |
| 다크 박스 텍스트 안 보임 | ❸ color 미지정 | 하위 태그별 `color:#fff` |
| 카드 간격 사라짐 | ⓫ `<br>` 스페이서 없음 | 카드 사이 `<br>` |
| `<details>` 안 접힘 | ⓬ 내부 빈줄/마크다운 | 빈줄 제거·HTML화 |
| 토글 삼각형 2개 | ⓭ 수동 삼각형 | summary에서 ▶/► 제거 |
| div 스타일 전부 무시 | ⓮ `<br>` + 블록 태그 동일 줄 | `<br>` 단독 줄 분리 |
| 레이아웃 붕괴 | ❶ 내부 빈줄 | 빈줄 전부 제거 |
