# HTML Div Style

**HTML div design system with 8 styles for Obsidian and HTML.**

Styled cards, headers, badges, note boxes. Default: Corporate Minimal (#1). Supports Obsidian `.md` (14 anti-break rules) and standalone `.html`.

### Example Prompts

```
"Apply style 3 to this table" → load style→wrap in divs→verify rendering
"Make a card layout" → select style→generate HTML cards
```

## Installation

```bash
git clone https://github.com/jasonnamii/html-div-style.git ~/.claude/skills/html-div-style
```
